¥textit{¥textbf{Beobachtung:} Stammfunktion ¥greenBox{ist nicht eindeutig}.}

Ist $F(x)$ Stammfunktion, so ist auch ¥greenBox{$F(x)+C$} fﾃｼr eine beliebige Konstante $C$ eine Stammfunktion.

$f(x)$ hat eine ganze Familie von Stammfunktionen: ¥definition{unbestimmtes Integral $¥int f(x) ¥, dx$}. (hier ist $f$ ¥definition{Integrand}).

¥subsection{Riemann-Integral (bestimmte Integrale)}
¥textbf{Idee:} Unterteile Funktion in $n$ (nicht zwingend gleich grosse) Teilintervalle, wﾃ､hle jeweils einen reprﾃ､sentativen Funktionswert in diesem Intervall, summiere alle einzelnen "Teilflﾃ､chen".

Danach $n ¥rightarrow ¥infty$, sodass die Lﾃ､nge der Teilintervalle gegen $0$ geht.

Sei $[a,b] ¥subset ¥mathbb{R}$ ein kompaktes Intervall.

¥definition{Zerlegung} von $[a,b]$ ist endliche Menge von Punkten $a=x_0 < x_1 < x_2 < ... < x_{n-1} < x_n = b$ mit $n ¥in ¥mathbb{N}$.

$x_i$ heissen ¥definition{Unterteilungspunkte.}

Zerlegung fﾃｼhrt zu ¥definition{Partition} des Intervalls $[a,b] = ¥{x_0¥} ¥cup (x_0, x_1) ¥cup ¥{x_1¥} ¥cup (x_1, x_2) ¥cup ¥dots ¥cup (x_{n-1}, x_n) ¥cup ¥{x_n¥}$.

Eine Zerlegung $a=y_0 < y_1 < ¥dots < y_{m-1} < y_m = b$ ist ¥definition{Verfeinerung} der Zerlegung $a=x_0 < x_1 < ¥dots < x_{n-1} < x_n = b$, falls gilt $¥{x_0, x_1, ¥dots, x_n¥} ¥subseteq ¥{y_0, y_1, ¥dots, y_n¥}$

$f: [a,b] ¥rightarrow ¥mathbb{R}$ ist ¥definition{Treppenfunktion}, falls $¥exists$ Zerlegung $a=x_0 < x_1 < ¥dots < x_n = b$ von $[a,b]$ sodass $¥forall k ¥in ¥{1, ¥dots, n¥}$ die Restriktionen von $f$ auf $(x_{k-1}, x_k)$, $f|_{(x_{k-1},x_k)}$ konstant sind.

$¥implies$ $f$ ist Treppenfunktion bzgl. der Zerlegung $a=x_0 < x_1 < ¥dots < x_n = b$.

Das ¥definition{Integral der Treppenfunktion} $f$ ﾃｼber $[a,b]$ ist die reelle Zahl $¥int_{a}^{b}f(x)¥, dx := ¥sum_{k=1}^{n}c_k(x_k - x_{k-1})$, wobei $c_k$ der Funktionswert von $f$ auf dem Intervall $(x_{k-1}, x_k)$ ist.

Sei $f:[a,b] ¥rightarrow ¥mathbb{R}$ eine Funktion. 
Dann ist die Menge der ¥definition{Obersummen $¥mathcal{U}(f)$}$:=¥{¥int_{a}^{b}s(x)¥, dx | s¥in ¥mathcal{TF}; s ¥geq f¥}$, 
respektive die Menge der ¥definition{Untersummen $¥mathcal{L}(f)$}$:=¥{¥int_{a}^{b}s(x)¥, dx | s¥in ¥mathcal{TF}; s ¥leq f¥}$
wobei $¥mathcal{TF}$ die Menge der Treppenfunktionen bezeichnet.

¥includegraphics[width=0.26¥columnwidth]{04_funktionen/obersummen_untersummen}

Eine beschrﾃ､nkte Funktion $f:[a,b] ¥rightarrow ¥mathbb{R}$ nennt man ¥definition{Riemann-integrierbar}, falls gilt $¥sup ¥mathcal{L}(f)=¥inf ¥mathcal{U}(f)$.

In diesem Fall wird der Wert das ¥definition{Riemann-Integral} von $f$ $¥int_{a}^{b} f ¥, dx := ¥sup ¥mathcal{L}(f)=¥inf ¥mathcal{U}(f)$ genannt.
$a$ nennt man ¥definition{untere Integrationsgrenze}, $b$ ¥definition{obere Integrationsgrenze} und $f$ den ¥definition{Integranden.}

¥theorem{Monotone Funktionen sind Riemann-integrierbar.}
¥textit{Monotone Funktionen sind in Intervall $[a,b]$ beschrﾃ､nkt.}

¥textit{Gilt auch, wenn Funktion auf manchen Teilintervallen monoton wachsend und auf manchen Teilintervallen monoton fallend ist.}

¥theorem{Stetige Funktionen sind Riemann-integrierbar.}
¥textit{mﾃｼssen aber nicht differenzierbar sein (z.B. Knickstellen ok)!}

¥textit{Gilt auch fﾃｼr Funktionen die ¥greenBox{in endlich vielen Stellen nicht monoton oder stetig} sind (wegen Aufteilung des Integrationsbereichs).}


¥subsubsection{Rechenregeln}
Seien $f,g:[a,b] ¥rightarrow ¥mathbb{R}$ zwei (Riemann-)integrierbare Funktionen.
¥begin{enumerate}
    ¥item ¥textbf{Linearitﾃ､t:} Seien zudem $¥alpha, ¥beta ¥in ¥mathbb{R}$. Dann ist auch $¥alpha f + ¥beta g$ integrierbar und es gilt
        $$¥int_{a}^{b} (¥alpha f + ¥beta g)(x) ¥,dx=¥alpha ¥int_{a}^{b} f(x) ¥, dx + ¥beta ¥int_{a}^{b} g(x) ¥, dx$$
    ¥item ¥textbf{Monotonie:} Falls gilt $f ¥leq g$, so gilt auch
        $$¥int_{a}^{b} f(x) ¥, dx ¥leq ¥int_{a}^{b} g(x) ¥, dx$$
    ¥item ¥textbf{Dreicksungleichung:} 
        $$|¥int_{a}^{b} f(x) ¥, dx| ¥leq ¥int_{a}^{b} |f(x)| ¥, dx$$
        ¥textit{Vorstellung: Positionsﾃ､nderung ist $¥leq$ zurﾃｼckgelegte Strecke}
    ¥item ¥textbf{Umkehrung der Integrationsrichtung:}
        $$¥int_{a}^{b} f(x) ¥, dx = - ¥int_{b}^{a} f(x) ¥, dx$$
        ¥textit{Vorstellung: Wir laufen wieder zurﾃｼck}
    ¥item ¥textbf{Aufteilung des Integrationsbereichs:}
        $$¥int_{a}^{b} f(x) ¥, dx = ¥int_{a}^{c} f(x) ¥, dx + ¥int_{c}^{b} f(x) ¥, dx, a ¥leq c ¥leq b$$
        ¥textit{Insb. interessant, wenn Integrationsbereich symmetrisch an Nullpunkt. Z.B. bei ungeraden Funktionen im Intervall $[-1, 1]$ ist Integral $0$, bei geraden Funktionen 2x der Wert von $[0,1]$.}
¥end{enumerate}

¥subsection{Hauptsatz}
¥theorem{Hauptsatz der Integral- und Differentialrechung:}

Sei $f:[a,b]¥rightarrow ¥mathbb{R}$ stetig. Dann ist fﾃｼr alle $C ¥in ¥mathbb{R}$ die Funktion $$F(x):=¥int_{a}^{x}f(y)¥,dy + C$$ eine Stammfunktion von $f$. Jede Stammfunktion int von dieser Form.
¥textit{(Konstruktion von Stammfunktionen)}

¥textit{
    ¥textbf{Beweis:} 
    $f$ stetig $¥implies$ $f$ auf jedem Intervall $[a,x], x ¥in [a,b]$ integrierbar.
    Dann zeigen, dass $F$ Stammfunktion von $f$ ist (mit $¥epsilon / ¥delta$-Stetigkeitsdefinition), $¥lim_{x ¥rightarrow x_0} |¥frac{F(x)-F(x_0)}{x-x_0} - f(x_0)|=¥lim |F'(x_0)-f(x_0)|=0$ (einzeln zeigen fﾃｼr $x ¥in (x_0, x_0+ ¥delta) ¥cap [a,b]$ und $x ¥in (x_0-¥delta, x_0) ¥cap [a,b]$).
}

Sei $F:[a,b]¥rightarrow ¥mathbb{R}$ stetig differenzierbar. Dann gilt fﾃｼr alle $x ¥in [a,b]$
$$F(x)= ¥underbrace{F(a)}_{¥textit{¥small{"Startwert"}}} + ¥underbrace{¥int_{a}^{x} F'(t) ¥, dt}_{¥textit{¥small{"Aufsummierung aller Verﾃ､nderungseffekte"}}}$$

¥textit{¥textbf{Herleitung:} Gem. Hauptsatz $F(x)=¥int_{a}^{x}F'(y)¥, dy + C, C ¥in ¥mathbb{R}$.
Insbesondere: $F(a)=¥underbrace{¥int_{a}^{a}F'(y)¥,dy}_{=0}+C$}

Sei $f:[a,b] ¥rightarrow ¥mathbb{R}$ stetig und $F:[a,b] ¥rightarrow ¥mathbb{R}$ Stammfunktion von $f$. Dann gilt
$$¥int_{a}^{b}f(t)¥,dt = F(x)|_a^b = F(b)-F(a)$$
¥textit{(Berechnung von bestimmten Integralen via Stammfunktion)}

¥greenBox{Verallgemeinerung fﾃｼr unstetige $f$}
Fﾃｼr eine Funktion $¥tilde{f}(y)$ mit endlich vielen Unstetigkeitsstellen $¥{x_1, ¥dots, x_n¥}$ ist $F(x):=¥int_{a}^{x}f(y)¥,dy+C$ noch eine Stammfunktion fﾃｼr $x ¥in [a,b] ¥ ¥{x_1, ¥dots, x_n¥}$

¥greenBox{Eindeutigkeit der Stammfunktion}
Der Hauptsatz liefert alle mﾃｶglichen Stammfunktion.

¥textit{¥textbf{Beweis:} Sei $G(x)$ eine weitere Stammfunktion, dann $H(x):=G(x)-¥int_{a}^{x}f(y) ¥, dy$. Dann gilt $H'(x)=f(x) - f(x)=0 ¥forall x ¥in [a,b]$. Dann muss aber $H(x)$ eine Konstante sein, d.h. $H(x)=C, C ¥in ¥mathbb{R}$, dies bedeutet $C = G(x)- ¥int_{a}^{x}f(y)¥,dy ¥iff G(x)= ¥int_{a}^{x}f(y)¥,dy + C, C ¥in ¥mathbb{R}$}
